## ESLint

不管是多人合作还是个人项目，代码规范都是很重要的。这样做不仅可以很大程度地避免基本语法错误，也保证了代码的可读性。

所以本项目中配置了ESLint，同理请把编辑器也安装此插件。你也可以设置[编辑器的ESLint快捷键](http://liqingsong.cc/article/detail/35)。
